Sou Carlos Henrique, desenvolvedor de software com dois anos de experiência em desenvolvimento web e um entusiasta de inteligência artificial. Concluí cursos em Web Fullstack, Ciência de Dados com Python, Redes de Computadores, Programação Web e Operação de Computadores.
